Groupings

  • service-ref

    • Description: Type of references to a particular service instance. This type can be used when defining module configuration to refer to a particular service instance. Containers using this grouping should not define anything else. The run-time implementation is expected to inject a reference to the service as the value of the container.
      • leaf: type

      • leaf: name
        • type: leafref
